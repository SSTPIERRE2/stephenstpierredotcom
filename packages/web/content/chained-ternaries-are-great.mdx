Check out these two functionally equivalent pieces of code, which do you like better?

```
let isShown = false;

if (conditionA && conditionB) {
  isShown = true;
} else if (conditionC) {
  isShown = true;
}

const isShown = conditionA && conditionB
  ? true
  : conditionC
  ? true
  : false;
```
